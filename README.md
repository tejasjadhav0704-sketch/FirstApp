## 🚀 How to Run

Download Git on your system

Clone the repository:

```bash
git clone https://github.com/tejasjadhav0704-sketch/FirstApp.git
```

Then:

1. Open in Android Studio  
2. Add Firebase `google-services.json`  
3. Sync Gradle  
4. Run the app
